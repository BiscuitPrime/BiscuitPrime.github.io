Tank A Faire

Tank A Faire is a 3D top-down arcade game made by the CELL (Video Game Making) club of Telecom Sudparis for the Mintech 2021 party. This game, made to be played in our club's very own (and self-made) arcade, pits you against a fellow player in the battlefield with a tank you can build. Choose between an array of cannons, turrets and tank chassis to create the perfect weapon to wipe out your adversary ! This game was made in Godot.
During this project I was the Lead (and only) 3D Artist. As such, I created all the various tank elements as well as the game's arena. The party's theme being Steampunk, the game itself reflects that with very Steampunk-y looking tanks (that I'm sure are very practical in real life). This project was very interesting as it was the first real group project that I made outside of school-mandated projects, with a clear deadline, client and the certainty that it would be played during the party. This was also the first project where I had to focus on creating meshs that both aligned design-wise and could be swapped along the player's choice. Furthermore, I had to create a design and creation pipeline to simplify the game integration part. You can find the schematics render of the tanks designed for the game below.
